# PalindroneCheckerApp
##Project Title
Palindrome Checker Application

##Objective
The objective of this project is to develop a simple Java-based application that checks whether a given word or sentence is a palindrome.

##Project Flow
Step 1: Start the Application
The program begins execution from the main() method.

Step 2: Take User Input

The user enters a word or sentence.
Input is stored in a String variable.

Step 3: Preprocess the Input
Convert the string to lowercase.
Remove spaces and special characters (if implemented).

Step 4: Reverse the String
Use a loop or StringBuilder to reverse the input.
Store reversed string in another variable.

Step 5: Compare Strings
Compare original processed string with reversed string.
If both are equal → It is a palindrome.
If not equal → Not a palindrome.

Step 6: Display Result
Print the result to the user.

##Conclusion
This project demonstrates basic Java programming concepts including:
String handling
Loops
Conditional logic
User interaction

It provides a foundation for understanding how real-world validation systems work.
